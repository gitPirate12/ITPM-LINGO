## 🚀 Project Overview

ITPM-LINGO is a web platform designed to support new Sinhala language learners through:

✅ **Language Tools**
- Real-time Sinhala translation
- Emoji-based language learning
- Collaborative forum

✅ **Core Features**
- JWT authentication system
- Thread-based discussion forum
- User profile management
- Interactive translation tools
